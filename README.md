# Team Project-MountainGO

#### 내용
- 등산의 즐거움을 붙일 수 있도록 도와주는 웹 사이트
- 지도를 통해 산의 위치와 정보를 알 수 있도록 하며, 등산 인증샷을 통해 성취감을 줄 수 있다.
#### 목표
- Jinja2 템플릿 엔진을 이용한 서버사이드 렌더링 사용
- JWT 인증 방식으로 로그인 구현
- 지도 API 사용, 산 마킹
- 스토리지를 이용한 사진 업로드
- 서버 배포
